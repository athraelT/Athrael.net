About the Gw2 Simulator

This Application is a personal project inspired by this reddit post and aims to simulate the math behind the post into a simple web based Precursor Simulator. As the project expands i will be adding more simulations which refer to famous (or infamous) in-game activities such as a Mystic Clover Simulator and so on... If you want to know more about how the Precursor Simulator works please visit the reddit post. As for the implementation, i used JQuery and Javascript as an attempt to get better at it.
